# 🔮 Gerador de Charadas

Um sistema de gerador de charadas com card interativo, sistema de pontuação e back-end em Flask com Firebase.

## ✨ Funcionalidades

- 🔄 **Card interativo** - Clique para virar e ver a resposta
- 🎲 **Charadas aleatórias** - Busca do Firebase via API
- 📊 **Sistema de pontuação** - Acertos, sequência e total
- 💾 **Salvamento automático** - Estatísticas salvas no navegador
- 🎨 **Design misterioso** - Tema escuro com detalhes em dourado
- 🔐 **Autenticação** - Login para rotas administrativas
- 📱 **Responsivo** - Funciona no celular e computador

## 🛠️ Tecnologias

**Front-end**
- HTML5
- TailwindCSS
- JavaScript

**Back-end**
- Python + Flask
- Firebase Firestore
- JWT para autenticação
